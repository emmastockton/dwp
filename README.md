# DWP

Author: Emma Stockton

## Requirements

- NodeJS (12)

Supports [nvm](https://github.com/nvm-sh/nvm), e.g. `nvm use`

## Setup

```
npm install
```

## Start the Service

```
npm start
```

Use the `PORT` environment variable to customise which port the service runs on, e.g. `PORT=8081 npm start`

## Run Tests

```
npm test
```
